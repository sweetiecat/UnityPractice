### Git 上傳
1. git clone 網址
2. 找到複製下來的資料夾
3. 將裡面的[.git][.gitignore]複製到unity資料夾裡<br>
3.5 將[.gitignore]複製到每個專案裡
4. cmd > cd unity的資料夾
5. git status看到專案名稱，但實際上傳只有專案裡的三個
6. git add .
7. git commit -m "隨便打"
8. git push
